960 grid system

There are three versions of 960 Grid System (12-columns, 16-columns and 24-columns). For the purposes of this theme we used 12-column system where the smallest section is 60px wide. You can see example of all columns you can use on illustration bellow. You can use any combination of the columns you want as long as the sum of their numbers is twelve (for example grid_8 and grid_4).
